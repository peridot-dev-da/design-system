Welcome to Peridot Design System

A comprehensive design system built with modern CSS and vanilla JavaScript for building consistent, accessible user interfaces.

Getting Started

Peridot Design System (PDS) is a lightweight, modern design system that provides:

  • Design Tokens - Consistent colors, typography, and spacing
  • Component Library - Reusable UI components with variants
  • Layout System - Responsive dashboard and page layouts
  • Theme Support - Built-in dark mode with CSS custom properties
  • Accessibility - WCAG compliant components with proper ARIA attributes
  • No Dependencies - Pure HTML, CSS, and JavaScript

Key Features

Responsive Sidebar

Collapses to overlay on mobile, fixed on desktop

Organized sections with expandable submenus

Sticky header with theme toggle and actions

Keyboard navigation and ARIA attributes

Design tokens, colors, typography, and utility classes

Basic Usage

Include the CSS and JavaScript

<!DOCTYPE html>
<html lang="en">
<head>
  <link rel="stylesheet" href="https://your-username.github.io/peridot-design-system/css/root.css">
  <script src="https://your-username.github.io/peridot-design-system/js/root.js"></script>
</head>
<body>
  <!-- Your content using PDS classes -->
  <button class="btn btn-primary">Primary Button</button>
</body>
</html>

Navigation States

Add the .active class to highlight the current page:

<a href="#current-page" class="nav-link active">
  <svg class="nav-icon">...</svg>
  Current Page
</a>

Try Dark Mode

Click the theme toggle in the header to switch between light and dark modes.
