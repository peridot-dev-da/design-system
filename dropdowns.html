JavaScript API

Control dropdowns programmatically with the JavaScript API.

  • Option 1
  • Option 2
  • Option 3
// Basic dropdown operations
pds.dropdown.show('#myDropdown');      // Show dropdown
pds.dropdown.hide('#myDropdown');      // Hide dropdown
pds.dropdown.toggle('#myDropdown');    // Toggle dropdown
pds.dropdown.hideAll();                // Hide all open dropdowns

// Create dropdown instance for advanced control
const dropdown = new pds.dropdown('#myDropdown', {
    autoClose: true    // Close when item is clicked
});

dropdown.show();
dropdown.hide();
dropdown.toggle();

// Event listeners
document.getElementById('myDropdown').addEventListener('dropdown.shown', (e) => {
    console.log('Dropdown shown:', e.detail.dropdown);
});

document.getElementById('myDropdown').addEventListener('dropdown.hidden', (e) => {
    console.log('Dropdown hidden:', e.detail.dropdown);
});

    Best Practices

    Usage Guidelines

    • Keep menus focused: Include only related actions to avoid overwhelming users
    • Use logical grouping: Group related items and separate with dividers
    • Provide clear labels: Use descriptive text and helpful icons
    • Handle disabled states: Show disabled items with clear visual indication

    Positioning

    • Default positioning: Dropdown appears below and aligned to the left
    • Smart positioning: Automatically adjusts if there's insufficient space
    • Mobile considerations: Full-width dropdowns on small screens

    Accessibility

    • Keyboard navigation with arrow keys and Enter/Space
    • Proper ARIA attributes for screen readers
    • Focus management when opening/closing
    • ESC key closes dropdown menus
